# react-baseline

### `How to run this project :`

1. git clone git@github.com:harshvardhanpatil/react-baseline.git
2. cd react-baseline
3. npm install
4. npm start

### `How to build :`

1. npm run build

### `Port And Details :`

Runs the app in the development mode.
Open http://localhost:3000 to view it in the browser.
